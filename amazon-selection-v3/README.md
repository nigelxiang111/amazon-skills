# Amazon Selection v3 - 亚马逊选品分析系统

> 智能选品决策引擎 | 双数据源 | 5大分析模块 | 8维度评分

## 📋 项目简介

Amazon Selection v3 是一个专业的亚马逊选品分析技能包，集成**卖家精灵**和**Sorftime**双数据源，通过智能意图识别自动路由到5大核心分析模块，为卖家提供从市场调研到利润核算的全链路选品决策支持。

### 核心特性

- 🎯 **智能路由**：自动识别用户意图，精准匹配分析场景
- 🔄 **双引擎驱动**：卖家精灵优先 + Sorftime补充，数据更全面
- 📊 **8维度评分**：市场规模、竞争强度、利润空间等综合评估
- 💰 **三情景分析**：保守/正常/乐观三种利润预测
- 🚀 **智能缓存**：Session级数据复用，提升响应速度
- 🌐 **多站点支持**：US/UK/DE/JP/CA等主流站点

## 📊 5大核心模块

### 1️⃣ 市场侦察 (Market Scout)

**适用场景**：评估某个类目/市场是否值得进入

**分析维度**：
- 市场规模（月销量/销售额）
- 竞争强度（商品数/卖家数/品牌集中度）
- 价格分布（均价/价格区间）
- 新品机会（新品占比/新品销量占比）
- 生命周期判断（导入期/成长期/成熟期）

**输出内容**：
- 市场概况卡片
- Top100产品分析
- 生命周期判断
- 进入建议（✅推荐 / ⚡谨慎 / ❌不建议）

### 2️⃣ 竞品解剖 (Product Spy)

**适用场景**：深度分析某个ASIN或竞品

**分析维度**：
- 销量趋势（月销量/增长率/季节性）
- 价格策略（售价/促销/价格波动）
- 流量来源（核心关键词/自然排名/广告词）
- 用户反馈（评分/评论/差评分析）
- 竞争优势（卖点/痛点/改进空间）

**输出内容**：
- 产品健康度评分
- SWOT分析
- 流量词TOP20
- 差评痛点提取
- 改进建议

### 3️⃣ 蓝海发现 (Niche Finder)

**适用场景**：寻找低竞争高潜力的新品机会

**筛选条件**：
- 月销量：1000-5000（有需求但未饱和）
- 评论数：<500（竞争门槛低）
- 价格区间：15-50美元（利润空间合理）
- 新品占比：>20%（市场接受新品）

**输出内容**：
- 蓝海类目TOP10
- 机会评分排序
- 进入难度评估
- 推荐理由

### 4️⃣ 利润实验室 (Profit Lab)

**适用场景**：精确核算产品利润和ROI

**计算维度**：
- 成本结构（采购/物流/FBA/广告）
- 三情景分析（保守/正常/乐观）
- 盈亏平衡点
- ROI和回本周期

**输出内容**：
- P&L利润表
- 成本占比饼图
- 敏感性分析
- 定价建议

### 5️⃣ 综合评估 (Full Evaluation)

**适用场景**：做最终决策前的全面评估

**整合内容**：
- 市场侦察结果
- 竞品解剖结果
- 利润核算结果
- 8维度综合评分

**输出内容**：
- 完整分析报告
- 风险提示
- 最终决策建议
- Excel数据导出

## 🔧 技术特性

### 双数据源协议

**优先级策略**：
```
Layer 1: Session缓存（已有数据直接复用）
Layer 2: 卖家精灵（数据质量优先）
Layer 3: Sorftime（补充+独有数据）
```

**自动降级**：
- 卖家精灵失败 → 自动切换Sorftime
- Sorftime失败 → 标注"数据暂不可用"
- 不重复采集同一维度数据

### 智能路由系统

**精确匹配**（置信度100%）：
- 包含ASIN码 → 竞品解剖
- 包含"算利润" → 利润实验室
- 包含"全面评估" → 综合评估

**语义匹配**（置信度80-95%）：
- 自动识别用户意图
- 多意图时展示选项
- 自动提取站点信息

### 8维度评分系统

| 维度 | 权重 | 说明 |
|------|------|------|
| 市场规模 | 20% | 月销量/销售额 |
| 竞争强度 | 15% | 商品数/卖家数 |
| 利润空间 | 20% | 毛利率/净利率 |
| 进入门槛 | 15% | 评论数/品牌集中度 |
| 增长潜力 | 10% | 增长率/新品占比 |
| 季节性 | 5% | 波动系数 |
| 供应链 | 10% | 1688采购难度 |
| 流量成本 | 5% | CPC/转化率 |

**评分标准**：
- 90-100分：✅ 强烈推荐
- 70-89分：⚡ 可以尝试
- 50-69分：⚠️ 谨慎评估
- <50分：❌ 不建议进入

## 📈 生命周期判断

| 阶段 | 新品占比 | 均评论数 | 特征 | 建议 |
|------|---------|---------|------|------|
| 导入期 | >30% | <100 | 市场刚起步，竞争少 | ✅ 最佳进入期 |
| 成长期 | >20% | 100-500 | 需求快速增长 | ✅ 黄金机会期 |
| 成熟前期 | 10-20% | 300-800 | 竞争加剧 | ⚡ 需差异化 |
| 成熟后期 | <10% | >800 | 市场饱和 | ❌ 谨慎进入 |

## 📝 注意事项

### 数据准确性
- 数据来源于第三方平台，仅供参考
- 建议结合多个数据源交叉验证
- 实际销量可能存在±20%误差

### 使用建议
- 优先使用综合评估模块做最终决策
- 关注生命周期判断，避免进入成熟后期市场
- 利润核算时考虑汇率波动和季节性因素
- 定期更新数据，市场变化快

### 性能优化
- Session级缓存自动生效
- 避免短时间内重复查询相同数据
- 大批量分析建议分批执行
