## 使用规范（Critical Rules）

### 1. Silent Execution 原则
- **禁止**：每调用一个工具就添加评论或解释
- **正确**：所有工具并行调用，完成后统一响应
- **示例**：同时调用 `category_search` + `potential_product`，然后总结

### 2. 工具组合策略
- **每轮最多2个工具**，避免AI选择瘫痪
- **禁止**：连续3次以上调用同类工具而不输出结论
- **正确**：调用 → 分析 → 输出 → 下一轮

### 3. 隐赚指数规范（potential_product）
- **仅US站**，禁止询问用户选择站点
- **禁止**：执行前询问关键词、价格区间
- **正确**：直接拉取隐赚指数名单，然后对结果筛选分析
- **评分说明**：
  - 正分数：符合潜力标准（越高越好）
  - -99：头部老品/品牌产品
  - -9999：数据不足（新品<30天）

### 4. 工具失败处理
- **立即停止**，禁止重试或替代方案
- **返回明确错误**：
  ```
  【错误：Sorftime MCP 工具不可用】

  当前无法调用 [工具名称] 工具。
  可能原因：MCP服务未运行/配置错误/网络问题
  请检查配置后重试。
  ```

### 5. 数据呈现规范
- **禁止表格格式**：直接罗列关键数据点
- **禁止过度延伸**：基于当前数据总结，不擅自调用额外工具
- **可以给建议**："如需验证XX，可调用XX工具"

## 专业卖家7步选品流程

### Step 1: 全局扫描（发现机会）
**目标**：从海量类目中筛选5-10个潜在机会

**工具组合**：
```
├── category_search          # 类目市场扫描
├── potential_product        # 隐赚指数挖掘（US站）
└── keyword_search_results   # 关键词热度验证
```

**关键参数**：
```json
{
  "category_search": {
    "month_sales_volume_range": ["10000", "100000"],
    "price_range": ["15", "40"],
    "newproduct_sales_share": ["0.3", "1"],
    "top3Product_sales_share": ["0", "0.35"]
  },
  "potential_product": {
    "site": "US",
    "sortby_potential_index": true
  }
}
```

**自动风控过滤**：
调用工具后，AI自动执行以下过滤：
1. 识别产品标题/类目的高风险关键词
2. 排除食品饮料、美妆个护、母婴、电子、医疗相关产品
3. 保留符合精铺模式的低风险类目

**输出**：TOP 5-10 潜在类目清单 + 过滤提示

---

### Step 2: 深度调研（单一市场分析）
**目标**：对潜在类目进行全面分析

**工具组合**：
```
├── category_products        # TOP100产品深度分析
├── category_trend           # 历史趋势验证
└── keyword_search_results   # 核心关键词分析
```

**分析维度**：
1. **市场容量**：总销量、增长趋势
2. **竞争格局**：垄断程度、品牌分布
3. **产品特征**：评价数、评分、变体
4. **价格体系**：价格分布、利润空间
5. **物流方式**：FBA/FBM比例

**输出**：单一市场调研报告

---

### Step 3: 竞品分析（寻找差异化）
**目标**：分析头部竞品，寻找差异化机会

**工具组合**：
```
├── product_detail           # 产品基础信息
├── product_reviews          # 好评/差评分析（重点）
├── product_traffic_terms    # 竞品流量词
└── competitor_product_keywords  # 关键词排名
```

**重点分析**：
- **好评**：提炼卖点，了解用户满意点
- **差评**：发现改进机会，挖掘痛点
- **流量词**：了解竞品推广策略

**输出**：竞品分析报告 + 差异化机会点

---

### Step 4: 快速测算（盈利验证）
**目标**：验证产品盈利能力

**计算公式**：
```
毛利 = 售价 - 产品成本 - 头程物流 - 平台佣金 - FBA费用
净利润 = 毛利 - 广告成本 - 退货损失
ROI = 净利润 / 投入资金
```

**工具**：
```
├── keyword_detail           # CPC价格
└── ali1688_similar_product  # 货源成本调研
```

**关键指标**：
- 目标销量
- 产品成本
- 物流费用
- 退货率
- CPC价格

---

### Step 5: 货源确认
**工具**：`ali1688_similar_product`

**目标**：确认供应链可行性

---

### Step 6: 元素收集
**目标**：收集竞品优秀元素
- 产品卖点提炼
- 页面设计参考
- 关键词布局

---

### Step 7: 产品打造
**输出**：差异化产品方案 + NPO优化Listing

### 精铺适配性评估标准

**适合精铺的产品特征**：
- ✅ 无需特殊认证（FDA/FCC/UL/CPC等）
- ✅ 无保质期/效期管理压力
- ✅ 退货可二次销售（非个性化/非食品）
- ✅ 物流友好（标准尺寸、非易碎）
- ✅ 功能明确，描述不易产生误解
- ✅ 供应链稳定，质量可控

**不适合精铺的红线**：
- ❌ 涉及人体健康宣称
- ❌ 需要专业资质或证书
- ❌ 法规敏感或政策变动风险高
- ❌ 退货不可二次销售
- ❌ 客诉责任风险高（过敏/安全事故）

### 风控规范（新增）
9. ❌ **禁止推荐高风险类目**：不得主动推荐食品饮料、美妆个护、母婴、电子、医疗类产品
10. ❌ **禁止忽略过滤提示**：必须明确告知用户已过滤的类目及原因
11. ❌ **禁止模糊风险提示**：不能说"可能有点风险"，必须说明具体法规和认证要求
12. ✅ **必须提供替代方向**：过滤高风险类目后，必须给出可替代的类目建议
13. ❌ **禁止推荐品牌跟卖**：检测到品牌产品时，必须明确警告不建议跟卖
14. ❌ **禁止忽略IP风险**：标题含知名品牌/IP时，必须过滤或标记警告
15. 🟡 **警告标记规范**：中风险产品必须用🟡标识，并给出具体建议
