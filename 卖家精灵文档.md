# 卖家精灵 (SellerSprite) MCP 工具列表

本列表对卖家精灵开放平台的 API 进行了模块化梳理，方便在定义 MCP (Model Context Protocol) 协议时将其作为独立的 Skills/Tools 进行创建和编排。

## 模块 A：基础工具 (Base)
*用于账号鉴权、配额检查的基础功能。*

- **可用次数查询** (`check_api_visits`): 查询当前月份的可用 API 额度和已用次数。

---

## 模块 B：选品与竞品分析 (Product & Competitor Research)
*用于对亚马逊商品、类目、竞品进行多维度查询与数据预估。*

- **查竞品** (`competitor_research`): 根据条件查询竞品数据。
- **选产品** (`product_research`): 多维度条件筛选亚马逊爆款产品。
- **查产品类目** (`category_research`): 获取产品类目及节点数据。
- **ASIN 详情** (`asin_details`): 查询指定 ASIN 的基础信息及运营数据。
- **ASIN 优惠趋势** (`asin_promotion_trends`): 查询 ASIN 的价格与打折趋势。
- **ASIN 销量预测** (`asin_sales_forecast`): 预估指定 ASIN 的未来销量。
- **BSR 销量预测** (`bsr_sales_forecast`): 基于 BSR 排名反推预估销量。
- **商品趋势详情 / Keepa** (`product_trend_keepa`): 获取类似 Keepa 的历史趋势曲线。

---

## 模块 C：关键词与流量分析 (Keyword & Traffic)
*用于 Listing 优化、PPC 广告选词、流量溯源及大盘热度分析。*

- **关键词挖掘** (`keyword_mining`): 输入核心词，挖掘高相关度、高搜索量的长尾词。
- **关键词选品** (`keyword_product_research`): 基于关键词表现倒推选品市场。
- **关键词反查** (`reverse_asin_keywords`): 反查 ASIN 的主要流量词列表。
- **拓展流量词** (`expand_traffic_keywords`): 基于基础词库进一步拓展流量词。
- **查流量来源** (`traffic_source_flow`): 追踪关键词流向和商品主要流量入口。
- **流量词 / 关联流量统计** (`traffic_keywords_stats`): 统计关键词或关联位带来的流量占比。
- **出单词反查** (`reverse_order_keywords`): 挖掘特定 ASIN 产生实际购买的出单词。
- **ABA 数据选品 (按周/按月)** (`aba_data_research`): 获取亚马逊官方 Brand Analytics 搜索词表现。
- **谷歌趋势** (`google_trends`): 查询词汇在 Google 的全网搜索热度。

---

## 模块 D：市场与大盘分析 (Market Analysis)
*涵盖大盘各维度的深度拆解。建议在构建 MCP 时合成一个大 Skill `market_analysis`，通过参数 `type` 来区分具体查询的维度。*

- **选市场列表** (`market_list`): 查询不同细分市场的整体列表。
- **选市场统计分析** (`market_statistics`): 包含细分市场的各项深度统计，具体包含：
  - 商品集中度
  - 品牌集中度
  - 卖家集中度
  - 卖家所属地分布
  - 卖家类型分布
  - 商品需求趋势
  - 上架时间分布
  - 评分数/评分值分布
  - 价格分布
  - A+视频分布

---

## 模块 E：其他辅助工具 (Others)
*支持日常运营和合规化排查的辅助类工具。*

- **查评论** (`review_research`): 抓取与分析竞品买家秀与 Review。
- **图片文字识别** (`image_ocr`): 跨境电商场景的图片 OCR 解析。
- **全球商标库检索** (`global_trademark_search`): 包含商标的范围、详情、列表与统计，用于侵权规避与品牌检索。
